# Cloudbnb

Welcome to Cloudbnb! This is a full stack web application built with the MERN stack (MongoDB, Express.js, React.js, and Node.js). Cloudbnb is designed to be an Airbnb clone, featuring functionality for booking, viewing, and managing accommodations.

## Features

- **User Authentication**: Register and log in to manage your bookings and profile.
- **Accommodation Listings**: Browse and view available accommodations.
- **Booking System**: Book accommodations and view your reservations.
- **Search and Filter**: Search for accommodations based on location, price, and other criteria.
- **User Profiles**: Manage your profile, view your bookings, and update your information.

## Technologies Used

- **Frontend**: React.js, Bootstrap
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JWT (JSON Web Tokens)
- **Hosting**: Not Hosted yet. 

## Installation

To get started with Cloudbnb, follow these steps:

1. **Clone the repository:**

    ```bash
    git clone https://github.com/dhruvraj508/cloudbnb.git
    ```

2. **Navigate to the project directory:**

    ```bash
    cd cloudbnb
    ```

3. **Install node, and yarn in your venv/local:**
```
brew install node
npm install yarn
```

4. **Run the application:**
    To start the application, do:
    ```
    cd client
    ```
    and then,
    ```
    yarn dev
    ```

7. **Open the application in your browser:**

    Visit `http://localhost:3000` to view the application.
